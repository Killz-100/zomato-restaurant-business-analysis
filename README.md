# 🍽️ Zomato Restaurant Business Analysis

## Project Overview

This project analyzes 10,000+ restaurant records from Zomato using Python and Power BI.

The objective was to identify patterns in ratings, cuisines, pricing, customer engagement, online delivery, and restaurant performance.

---

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI

---

## Key Business Questions

1. Which cuisines are the most popular?
2. How are ratings distributed?
3. Which cities have the most restaurants?
4. How does online delivery affect restaurant performance?
5. What pricing trends exist across restaurant categories?

---

## Dashboard Pages

- Overview
- Ratings Analysis
- Cost Analysis
- Online & Booking Analysis
- Cuisine Intelligence

---

## Key Insights

- North Indian cuisine had the highest restaurant count.
- Most restaurants were rated between 3 and 4.
- Online delivery was widely adopted.
- Customer engagement varied significantly by location and cuisine.
